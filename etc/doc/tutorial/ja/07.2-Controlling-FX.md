# エフェクトの制御

エフェクトも、少々異なる方法をとりますが、制御する事が出来ます。

```
with_fx :reverb do |r|
 play 50
 sleep 0.5
 control r, mix: 0.7
 play 55
 sleep 1
 control r, mix: 0.9
 sleep 1
 play 62
end
```
変数を使用する代わりに、*do/end*プロックのゴールポストパラメータを使用します。”`|`”と”`|`”の間に、エフェクターを実行するためのユニークな名前を指定する必要があります。この動作は、パラメーター化された関数を使用する場合と同じです。

さあ、シンセやエフェクトをコントロールしてみよう！
